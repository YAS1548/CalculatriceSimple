Calculatrice développée en Java dans le cadre du projet de 
Programmation Orientée Objet (POO) en 2ème année Licence (L2) à l'USTHB.

Le projet applique les principes fondamentaux de la POO :
- Héritage et polymorphisme
- Classes abstraites (CalculMath, OperationBinaire, OperationUnaire)

Opérations supportées :
- Arithmétiques : Addition, Soustraction, Multiplication, Division
- Scientifiques : Sin, Cos, Exp, Log, Racine carrée (Sqrt)

 Contenu :
- Code source Java (package Calculatrice)
- Diagramme UML (UML CALCULATRICE.pdf)

 Prérequis :
- Java JDK 8 ou supérieur
- N'importe quel IDE Java (Eclipse, IntelliJ, VSCode)
